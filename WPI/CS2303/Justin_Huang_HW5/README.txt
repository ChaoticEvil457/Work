Based on provided file binary_tree_inC.c

This program creates, prints, and deletes a Binary Tree

In order to compile, run
g++ -Wall -o tree binary_tree_inCpp.cpp
then run ./tree
This program reads values from a file named data.txt and adds them to a Binary Tree data structure.
It then prints out the nodes in order.
After printing, the program deletes the tree, printing out the nodes as it removes them.
